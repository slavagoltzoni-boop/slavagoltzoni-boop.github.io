## Python

Самый популярный язык для науки о данных.

**Библиотеки:**
- NumPy — работа с массивами
- Pandas — анализ данных
- Matplotlib — визуализация
- Scikit-learn — машинное обучение

## R

Язык для статистики с мощной визуализацией (ggplot2).

## Julia

Современный язык для высокопроизводительных вычислений. Скорость C, синтаксис Python.

## MATLAB

Коммерческий пакет для инженерных расчётов и симуляций.

## Сравнение

| Язык | Скорость | Простота | Цена |
|------|----------|----------|------|
| Python | Средняя | Высокая | Бесплатно |
| R | Средняя | Средняя | Бесплатно |
| Julia | Высокая | Высокая | Бесплатно |
| MATLAB | Высокая | Средняя | Платно |
