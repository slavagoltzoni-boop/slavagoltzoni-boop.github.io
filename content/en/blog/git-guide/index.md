## What is Git?

Git is a distributed version control system.

## Essential Commands

| Command | Purpose |
|---------|---------|
| git init | Create repository |
| git add . | Stage files |
| git commit -m | Save changes |
| git push | Upload to server |
| git pull | Download updates |
