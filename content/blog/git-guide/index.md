## Что такое Git?

Git — это распределённая система управления версиями. Она позволяет:
- Сохранять историю изменений кода
- Работать над проектом в команде
- Откатывать изменения при ошибках

## Основные команды

| Команда | Назначение |
|---------|------------|
| git init | Создать репозиторий |
| git add . | Добавить файлы |
| git commit -m | Зафиксировать изменения |
| git push | Отправить на сервер |
| git pull | Получить обновления |

## Типичный workflow

1. git pull — получить актуальную версию
2. Внести изменения в файлы
3. git add . — добавить в индекс
4. git commit -m "описание" — зафиксировать
5. git push — отправить на GitHub
