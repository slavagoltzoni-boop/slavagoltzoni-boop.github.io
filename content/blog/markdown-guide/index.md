## Что такое Markdown?

Markdown — это лёгкий язык разметки, созданный для удобного написания и чтения текста. Он используется в GitHub, Hugo, документации и блогах.

## Базовый синтаксис

### Заголовки

# H1
## H2
### H3

### Форматирование текста

- **Жирный текст**
- *Курсив*
- ~~Зачёркнутый~~

### Списки

1. Нумерованный список
2. Второй пункт

- Маркированный список
- Ещё пункт

## Преимущества Markdown

- Простой и читаемый синтаксис
- Поддерживается везде (GitHub, Hugo, Notion)
- Легко конвертируется в HTML
